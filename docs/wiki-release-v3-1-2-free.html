Versão 3.1.2 FREE: primeira release multi-plataforma

Evolução da linha FREE do CaraCore-PDV no canal público da loja. Primeira release com suporte oficial a Windows, Linux e macOS. Pacote portátil ZIP por plataforma — requisito: Java 21+ instalado no sistema.

Versão FREE 3.1.2 — lançada em 05/2026. Esta é a oferta pública canônica atual da loja para teste real. Agora disponível para Windows, Linux e macOS. Para ampliar operação, integrar PIX no sistema e adicionar mais estrutura comercial, consulte a edição PREMIUM.

O que foi adicionado nesta versão

  • Multi-plataforma: primeira release com pacote portátil para Windows, Linux e macOS — a promessa de "trilha de evolução para Linux e macOS" da v3.1.1 foi entregue.
  • Workflow CI/CD corrigido: pipeline de release reescrito com empacotamento direto do Quarkus fast-jar, eliminando dependência do Electron (não rastreado no repositório).
  • Script de inicialização multiplataforma: iniciar-pdv.bat (Windows) e iniciar-pdv.sh (Linux/macOS) incluídos em cada ZIP.
  • SHA256 por plataforma: cada ZIP tem hash individual em SHA256SUMS.txt e RELEASE_MANIFEST.json.
  • Integridade de entrega: script delivery_free_multiplatform_to_releases.py na oficina para publicação controlada e rastreável.

Instalação rápida

  1. Certifique-se de ter Java 21+ instalado: java -version
  2. Baixe o pacote da sua plataforma na página de download ou no canal interno de instaladores.
  3. Extraia o ZIP para uma pasta local (ex.: C:\Programas\CaraCore-PDV-FREE no Windows).
  4. Windows: execute iniciar-pdv.bat. Linux/macOS: execute ./iniciar-pdv.sh.
  5. Aguarde ~10 segundos para o backend Quarkus iniciar.
  6. Acesse o sistema no navegador em http://localhost:8765 ou pela janela principal do aplicativo.

Primeiro login: usuário admin · senha admin · troca obrigatória no primeiro acesso.

Dados: banco SQLite local criado automaticamente em %APPDATA%\caracore\ (Windows) ou ~/.config/CaraCore PDV/ (Linux/macOS).

Integridade do pacote

Cada release inclui SHA256SUMS.txt e RELEASE_MANIFEST.json. Para validar:

# Windows (PowerShell)
Get-FileHash caracore-pdv-v3.1.2-free-windows-x64.zip -Algorithm SHA256

# Linux
sha256sum caracore-pdv-v3.1.2-free-linux-x64.zip

# macOS
shasum -a 256 caracore-pdv-v3.1.2-free-macos-x64.zip

Compare com o hash correspondente em SHA256SUMS.txt da mesma release.
