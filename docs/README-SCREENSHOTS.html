Manual de Treinamento - Guia de Screenshots

Guia operacional para captura de telas de treinamento do CaraCore-PDV.

Observação: esta página é material técnico de apoio. A oferta pública atual da loja continua concentrada na edição v3.1.1-free.

Como usar

  1. Compile o projeto: mvn clean compile -DskipTests.
  2. Execute a aplicação JavaFX: mvn javafx:run.
  3. Navegue pelas telas e capture imagens em PNG.

Captura automática (opcional)

Use o script:

python scripts/capturador-screenshots.py --backend auto --upscale 2

Telas principais

  • Login
  • Frente de Caixa
  • Confirmação de Pagamento
  • Cadastro de Produto
  • Pesquisa de Cliente
