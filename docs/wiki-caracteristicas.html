Características completas da aplicação

Baseado na garagem da aplicação D:\dev\caracore-pdv, este guia consolida as capacidades atuais do CaraCore-PDV em linguagem operacional.

Fonte oficial: código e documentação da garagem. Escopo desta página: produto, arquitetura, módulos, operação, segurança, testes e planos.

1. Visão de produto

  • PDV Selo Verde para varejo brasileiro com foco em operação antifrágil
  • Execução local com SQLite para soberania de dados e continuidade de venda
  • Fluxo preparado para Reforma Tributaria 2026-2033
  • Distribuicao principal em Windows Desktop (EXE) com abertura em localhost:8080

2. Stack tecnológica

Camada Tecnologias Observação
Backend Java 21, Spring Boot, Spring Data JPA, Spring Security, Hibernate Base de negócio e APIs REST do PDV
Frontend Thymeleaf, Bootstrap, HTMX, ApexCharts, jQuery Interface web responsiva para operação e administração
Banco SQLite Perfil padrão e armazenamento local
Desktop Electron + JRE embarcada Instalador Windows com bootstrap do servidor local
Relatórios JasperReports Geracao de PDF gerencial e fiscal

3. Módulos funcionais

  • PDV frente de caixa com atalhos, fluxo rápido e formas de pagamento
  • Gestão de vendas com ciclo completo e rastreio de status
  • Cadastro e pesquisa de produtos, clientes, vendedores, lojas e operadores
  • Controle de estoque por loja e monitoramento administrativo
  • Dashboard executivo com indicadores em tempo real
  • Relatórios gerenciais e relatório para contador (plano Premium)
  • Suporte a operação multi-loja

4. Operação local e contingência

  • Banco local em SQLite para operar mesmo com internet instável
  • Modo de contingência com fila local para eventos fiscais
  • Fluxo antifrágil para manter venda em condição degradada
  • Desktop com inicialização assistida do backend local quando necessário
  • Primeiro acesso de degustação na loja: admin / 123456

5. Segurança e hardening

  • Controles de autenticação e autorização na aplicação
  • Empacotamento desktop com foco em integridade e proteção de código
  • Ofuscação no fluxo de distribuição
  • Diagnóstico guiado no desktop com logs e coleta de evidência
  • Trilha de suporte com canais formais (e-mail, WhatsApp, Telegram)

6. Fiscal, compliance e sustentabilidade

  • Motor fiscal com evolução aderente ao cronograma 2026-2033
  • Recibo digital para reduzir custo de papel e impressão
  • Registro local e rastreabilidade de operações
  • Posicionamento Selo Verde com foco em economia operacional

7. Planos e capacidades

Plano Escopo Recursos destacados
Free (Degustação) Validação real da operação 100 vendas para teste, operação local, fluxo base do PDV
Premium Continuidade operacional ampliada Recuperação de senha por SMS, relatório do contador, suporte prioritario e automações premium

8. Testes e qualidade

  • Testes Java (unitários e integração) com JUnit e Mockito
  • Cobertura com JaCoCo
  • Testes de interface com Selenium e Pytest
  • Testes fiscais no desktop com Vitest e mocks da SEFAZ
  • Validações de contingência e sincronização em fila local

9. Governança de release local

  • Canal de distribuição oficial para o cliente: pdv.caracore.com.br/installers/
  • Canal estável para produção e canal pré-release para validações controladas
  • Integridade do instalador validada em pipeline antes de publicação
  • A loja concentra vitrine, wiki e binários servidos no mesmo domínio
  • Rastreabilidade por tag de versão e histórico público de release
