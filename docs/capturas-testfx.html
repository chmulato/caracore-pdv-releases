TestFX - Captura Automatizada de Screenshots

Fluxo automatizado para captura consistente das telas JavaFX de treinamento do CaraCore-PDV.

Observação: este conteúdo é técnico e de apoio interno para documentação e treinamento.

Execução

mvn clean test -Dtest=ScreenshotsTrainingTestFX

Execuções específicas

mvn clean test -Dtest=ScreenshotsTrainingTestFX#testCaptureTelaLogin
mvn clean test -Dtest=ScreenshotsTrainingTestFX#testCaptureFrenteCaixa

Saída esperada

As imagens são geradas em pastas versionadas dentro de screenshots/.

Observações

  • Executar com JDK 21.
  • Manter escala de tela em 100% para fidelidade visual.
  • Aguardar renderização entre navegações quando necessário.
