Visao geral

Esta wiki fica dentro da propria loja do CaraCore-PDV e concentra instrucoes curtas para validacao da release e primeiro uso em ambiente Windows.

  • Canal atual: instalador Windows x64
  • Perfil: degustacao Free com SQLite local
  • Atualizacao automatica suportada por artefatos auxiliares

Primeiro acesso

  • URL local: http://localhost:8080
  • Usuario inicial: admin
  • Senha inicial: 123456
  • Banco local: SQLite

Checklist rapido pos-instalacao (Windows)

  • Confirmar abertura do app local em http://localhost:8080
  • Validar login admin / 123456 no ambiente de degustacao
  • Cadastrar 1 produto, registrar 1 venda e emitir 1 recibo digital de teste
  • Fechar e reabrir o app para validar persistencia no SQLite local
  • Verificar pasta de logs do desktop em caso de comportamento inesperado

Canal de releases e politica de atualizacao

  • Canal estavel: sempre pelo link releases/latest
  • Canal pre-release: usado para validacoes de campo antes da proxima final
  • Fonte unica: distribuicao oficial apenas no GitHub Releases (sem espelho local de instalador)
  • Boa pratica: em ambiente produtivo, priorizar release estavel
Abrir estavel mais recente Historico de releases

Diagnostico rapido (suporte N1)

  • App nao abre: reiniciar Windows e executar novamente como usuario local
  • Tela em branco: validar se localhost:8080 responde no navegador
  • Porta ocupada: encerrar processos antigos do app e relancar
  • Falha apos atualizacao: voltar para a release estavel no canal oficial
  • Persistindo erro: enviar evidencia com horario e captura para suporte oficial
