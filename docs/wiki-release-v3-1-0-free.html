Versão 3.1.0 FREE: edição gratuita multi-plataforma

Primeira entrega da linha FREE do CaraCore-PDV. Disponível para Windows, Linux e macOS via pacote portátil Electron + Quarkus JVM. Ideal para avaliação, microempresas e operações de baixo volume.

Versão FREE 3.1.0 — lançada em 05/2026. Edição gratuita com funcionalidades essenciais de ponto de venda. Para operações completas com NF-e, PIX e múltiplos operadores, consulte a edição PREMIUM.

O que foi adicionado nesta versão

  • Edição FREE oficial: primeiro empacotamento multi-plataforma da linha gratuita, com limites claros e operação offline-first.
  • Suporte a Windows, Linux e macOS: workflow CI/CD (release-free.yml) produz ZIPs portáteis para as três plataformas a partir de uma única base de código.
  • Motor Quarkus 3.x JVM (fast-jar): engine construída uma vez no CI e distribuída para todos os pacotes — sem necessidade de compilação nativa por plataforma.
  • Shell Electron cross-platform: detecção automática do runner nativo (*-runner) ou JVM (quarkus-run.jar) em Linux e macOS, além do Windows.
  • Identificação visual FREE: título da janela exibe CaraCore PDV FREE para diferenciar da edição paga.

Instalação rápida

  1. Baixe o ZIP correspondente à sua plataforma na página de download ou no GitHub Releases.
  2. Extraia o ZIP para uma pasta local (ex.: C:\Programas\CaraCore-PDV-FREE).
  3. Windows: execute CaraCore PDV.exe. Linux: execute caracore-pdv. macOS: abra CaraCore PDV.app com botão direito → Abrir.
  4. Aguarde ~10 segundos para o backend Quarkus iniciar (progresso visível no console).
  5. Acesse o sistema no navegador em http://localhost:8765 ou pela janela Electron.

Primeiro login: usuário admin · senha admin · troca obrigatória no primeiro acesso.

Dados: banco SQLite local criado automaticamente em %APPDATA%\caracore\ (Windows) ou ~/.config/CaraCore PDV/ (Linux/macOS).

Integridade do pacote

Cada release inclui SHA256SUMS.txt e RELEASE_MANIFEST.json. Para validar no Windows:

Get-FileHash caracore-pdv-v3.1.0-free-windows-x64.zip -Algorithm SHA256

No Linux/macOS:

sha256sum caracore-pdv-v3.1.0-free-linux-x64.zip

Compare com o hash em SHA256SUMS.txt da mesma release.
