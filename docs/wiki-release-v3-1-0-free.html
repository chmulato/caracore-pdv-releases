Instalação rápida

  1. Baixe o pacote Windows na página de download ou use o canal interno de instaladores.
  2. Extraia o ZIP para uma pasta local (ex.: C:\Programas\CaraCore-PDV-FREE).
  3. Windows: execute CaraCore PDV.exe.
  4. Aguarde ~10 segundos para o backend Quarkus iniciar (progresso visível no console).
  5. Acesse o sistema no navegador em http://localhost:8765 ou pela janela principal do aplicativo.

Primeiro login: usuário admin · senha admin · troca obrigatória no primeiro acesso.

Dados: banco SQLite local criado automaticamente em %APPDATA%\caracore\ (Windows) ou ~/.config/CaraCore PDV/ (Linux/macOS).

Integridade do pacote

Cada release inclui SHA256SUMS.txt e RELEASE_MANIFEST.json. Para validar no Windows:

Get-FileHash caracore-pdv-v3.1.0-free-windows-portable.zip -Algorithm SHA256

Compare com o hash em SHA256SUMS.txt da mesma release.
