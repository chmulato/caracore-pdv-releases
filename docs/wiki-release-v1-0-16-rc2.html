Release v1.0.16-rc2

Pré-release avançada da série 1.0.16 com foco em confiança operacional para o cliente.

Publicado em 04/04/2026. Release candidata para reforçar estabilidade de instalação, atualização e operação da loja.

Resumo para o cliente

  • Perfil: release candidate para homologação de campo.
  • Foco: garantir operação sem surpresas no caixa.
  • Entrega: instalador oficial para Windows 10/11.
  • Modelo: dados locais para continuidade do negócio.

Informações da aplicação

Domínio Definição na v1.0.16-rc2
Modelo PDV local para Windows com acesso via localhost:8080
Persistência SQLite local para continuidade operacional
Distribuicao Instalador EXE com suporte a atualização incremental
Objetivo da RC Validar instalação, fluxo mínimo de venda e estabilidade geral

Regras de negócio centrais

  • RB-01 Operação local: a venda deve ser concluível com banco SQLite local.
  • RB-02 Fonte oficial: instalador e metadados de update devem vir do canal oficial de releases.
  • RB-03 Integridade de validação: toda anomalia em homologação deve ser registrada com evidência.
  • RB-04 Fluxo mínimo obrigatorio: login, cadastro de item, venda e recibo devem funcionar sem erro.

Conteúdo publicado

  • Instalador .exe
  • Arquivo builder-debug.yml
  • Arquivo .blockmap para atualização incremental
  • Guia INSTALACAO.md

Destaques desta RC2 (visão de negócio)

  • Degustação Free com capacidade real de validação em loja.
  • Operação local preparada para manter atendimento contínuo.
  • Distribuição mais segura e padronizada para implantação.
  • Base de atualização pronta para evolução controlada do produto.
  • Maior confiança para transição para versões finais.

Funcionalidades por módulo

1) Acesso e usuários

  • Login administrativo para bootstrap de ambiente.
  • Base para governança de operadores.

2) Cadastros e venda

  • Cadastro de itens e execução de ciclo mínimo de venda.
  • Persistência local dos dados após reinício.

3) Recibo e operação

  • Emissão de comprovante digital no fluxo de teste.
  • Comportamento orientado a continuidade operacional local.

Fechamento GERENTE / ADMIN Conferir consistência de vendas e estoque. Registrar anomalias de homologação com evidência.

Critérios de validação desta versão

  • Instalação sem erro em Windows 10/11
  • Subida da aplicação em localhost:8080
  • Fluxo mínimo validado ponta a ponta
  • Dados preservados no SQLite após reinicialização
