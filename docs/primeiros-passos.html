Guia Rápido de Primeiros Passos

Este guia orienta a instalação e autorização inicial do CaraCore-PDV em Windows.

Contexto da loja: a oferta pública atual para validação é a edição v3.1.1-free. Este documento é material técnico de apoio para implantação e suporte.

Pré-requisitos

  • Windows 10 ou 11
  • Python 3.8+ (necessário no instalador script)

1. Executar instalador

python scripts\install_windows.py

2. Se o Defender bloquear

Abra o PowerShell como Administrador e execute:

Add-MpPreference -ExclusionPath "$env:APPDATA\CaraCore"

3. Se o SmartScreen bloquear

  • Clique em More info e depois em Run anyway.
  • Ou abra propriedades do executável e marque Unblock, quando disponível.

4. Verificação pós-instalação

  • Arquivos principais: %APPDATA%\CaraCore\
  • Confirme java_home no config.json
  • Telemetria: %APPDATA%\CaraCore\telemetry.log

5. Timeout de backend

Quando ocorrer BACKEND_TIMEOUT, valide rede, processo Java e logs locais.

6. Suporte

Se persistir, envie o arquivo %APPDATA%\CaraCore\telemetry.log para o suporte técnico.
