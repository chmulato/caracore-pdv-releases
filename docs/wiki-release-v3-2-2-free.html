Versão 3.2.2 FREE: Java 25 + Alinhamento PDV Rust→Java

Evolução da linha FREE do CaraCore-PDV. Runtime atualizado para Java 25; alinhamento completo com a referência PDV Rust (navegação por zonas, InactivityLock, status bar, supervisor elevation e SetupGate). Multi-plataforma: Windows, Linux e macOS. Requisito: Java 25+ instalado no sistema.

Versão FREE 3.2.2 — lançada em 06/2026. Esta é a oferta pública canônica atual da loja para teste real. Disponível para Windows, Linux e macOS com Java 25. Para ampliar operação, integrar PIX no sistema e adicionar mais estrutura comercial, consulte a edição PREMIUM.

O que foi adicionado e alterado nesta versão

  • Java 25: runtime, compilação e JavaFX atualizados para Java 25 (Temurin 25). Requisito mínimo do usuário final passa a ser Java 25+.
  • Zonas de navegação por domínio: bottom nav e centro de configurações renomeados por domínio — "Gestão da loja", "Operação", "Caixa e turno", "Financeiro", "Fiscal e auditoria", "Suporte".
  • InactivityLock: bloqueio automático de tela por inatividade configurável — segurança operacional no balcão.
  • PdvStatusBar: barra de status operacional sempre visível com conexão, loja, operador, turno e modo offline/online.
  • Supervisor elevation: botões de desconto e cancelamento sempre visíveis na Nova Venda — habilitados por perfil ou via autorização inline de supervisor.
  • SetupGate: tela de configuração inicial bloqueante — o sistema não avança até loja e operador estarem configurados corretamente.
  • Scripts Python: health-check e validação migrados para Python (pasta scripts/) — padronização com o ecossistema.
  • Pipeline CI: workflow release-free.yml atualizado para java-version: '25' (Temurin 25).

Para que perfil de loja esta versão faz sentido

  • Alta aderência: pequena operação com 1 a poucos caixas, foco em venda rápida e necessidade de trabalhar mesmo com internet instável.
  • Aderência média: loja em fase de transição de processo, que precisa provar fluxo e rotina antes de ampliar estrutura.

  • Baixa aderência: operação que já exige integração fiscal completa, múltiplas integrações legadas e automações além do escopo Free.

Instalação rápida

  1. Certifique-se de ter Java 25+ instalado: java -version
  2. Baixe o pacote da sua plataforma na página de download.
  3. Extraia o ZIP para uma pasta local (ex.: C:\Programas\CaraCore-PDV-FREE no Windows).
  4. Windows: execute iniciar-pdv.bat. Linux/macOS: execute ./iniciar-pdv.sh.
  5. Aguarde ~10 segundos para o backend Quarkus iniciar.
  6. Acesse o sistema no navegador em http://localhost:8765 ou pela janela principal do aplicativo.

Primeiro login: usuário admin · senha admin · troca obrigatória no primeiro acesso.

Dados: banco SQLite local criado automaticamente em %APPDATA%\caracore\ (Windows) ou ~/.config/CaraCore PDV/ (Linux/macOS).

Integridade do pacote

Cada release inclui SHA256SUMS.txt e RELEASE_MANIFEST.json. Para validar:

# Windows (PowerShell)
Get-FileHash caracore-pdv-v3.2.2-free-windows-x64.zip -Algorithm SHA256

# Linux
sha256sum caracore-pdv-v3.2.2-free-linux-x64.zip

# macOS
shasum -a 256 caracore-pdv-v3.2.2-free-macos-x64.zip

Compare com o hash correspondente em SHA256SUMS.txt da mesma release.
