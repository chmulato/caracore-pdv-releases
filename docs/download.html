Versão de Degustação (Free) • Windows 10/11

Canal atualizado: release estável 2.0.0 e pré-release 2.0.1-rc1. A versão 3 (branch java_21) já está apresentada na wiki com foco em benefícios para o cliente: continuidade de vendas, mais controle e preparo para 2026.

O CaraCore-PDV é distribuído como instalador Windows (EXE). Ao abrir, o servidor sobe em localhost:8080. Banco apenas SQLite — soberania local, operação antifrágil, sem necessidade de PostgreSQL.

Condições da Degustação (Free):

  • 100 vendas completas para teste real
  • Operação offline (SQLite)
  • PIX com conferência manual

Login na degustação (ambiente local Windows, localhost): usuário admin, senha 123456.
